盛世公主．打卡趣

活動期間：2017 / 11 / 01 － 11 / 15
活動步驟:

#盛世公主遊你夢想，即日起至11/15(三)，選出喜愛的盛世公主號特色設施，Tag一位朋友並寫下為什麼想在這裡打卡分享，同時Hashtag #盛世公主遊你夢想，公開分享至個人Facebook塗鴉牆，就有機會抽到盛世公主號沖繩４日遊。

  1. 至《盛世公主‧打卡趣》專區，選擇喜愛的盛世公主號特色設施
  2. 填寫參加者個人基本資料
  3. 選出想進行打卡的設施，Tag一位朋友並寫下為什麼想在這裡打卡分享，同時Hashtag #盛世公主遊你夢想
  4. 公開分享至個人Facebook塗鴉牆即有抽獎資格
獎項：
  1. 2017 / 11 / 08、2017 / 11 / 16分別抽出100名獲贈公主紀念禮組；於2017 / 11 / 13、2017 / 11 / 20公布得獎資料。 (每帳號每日限參加一次，11/08統計範圍：2017 / 11 / 01－11 / 07。11/16統計範圍：2017 / 11 / 08 - 11 / 15)
  2. 2017/11/24抽出「2018/4/8 基隆港出發，盛世公主號4天沖繩自主遊_雙人內艙房」一組 (每帳號每日限參加一次)
活動結束

公主時刻．遊你夢想

活動期間: 2017 / 11 / 17 － 11 / 30
活動步驟:

#盛世公主遊你夢想，即日起至11/30(四)，選出喜愛的夢想短片，Tag三位朋友並寫下「2018一起搭盛世公主號，實現我們的夢想時刻吧！」，同時Hashtag #盛世公主遊你夢想，公開分享至個人Facebook塗鴉牆，就有機會抽到盛世公主號沖繩４日遊。

  1. 至《公主時刻‧遊你夢想》觀賞夢想短片
  2. 填寫參加者個人基本資料
  3. 選出喜愛的夢想短片，Tag三名朋友並寫下「2018一起搭盛世公主號，實現我們的夢想時刻吧！」，同時Hashtag #盛世公主遊你夢想
  4. 公開分享至個人Facebook塗鴉牆及有抽獎資格
